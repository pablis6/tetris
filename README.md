# Tetris

Este proyecto es una implementación sencilla del juego Tetris utilizando JavaScript Vanilla y Canvas.

## Estructura de archivos

El proyecto tiene la siguiente estructura de archivos:

- `src/index.js`: Este archivo es el punto de entrada de la aplicación. Aquí puedes escribir el código JavaScript para inicializar y controlar el juego Tetris.

- `src/canvas.js`: Este archivo contiene el código JavaScript para manipular el elemento canvas y dibujar los elementos del juego, como las piezas y el tablero.

- `src/styles.css`: Este archivo contiene los estilos CSS para darle formato y diseño al juego Tetris.

- `index.html`: Este archivo es la página HTML principal del juego Tetris. Aquí puedes enlazar los archivos JavaScript y CSS, y también puedes agregar el elemento canvas donde se dibujará el juego.

## Ejecución del juego

Para ejecutar el juego Tetris, simplemente abre el archivo `index.html` en tu navegador web. Asegúrate de tener una conexión a Internet activa, ya que se utilizan algunas fuentes y estilos externos.

## Puntuacion
1 línea: 10 puntos × número de nivel en el que te encuentras.
2 líneas: 25 puntos × nivel.
3 líneas: 40 puntos × nivel.
4 líneas: 60 puntos × nivel.
Por cada pieza colocada: 1 punto × nivel.
